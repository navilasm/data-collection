# Data Collection Repository

A curated collection of datasets gathered from various sources.
This repository contains **data only** — no code, scripts, or processing pipelines.

## Overview

This repository serves as a centralized place to store and organize datasets for:

* Data analysis
* Machine learning
* Research and academic use
* Data exploration/visualization projects

All datasets are stored in structured formats such as:

* CSV
* JSON
* Excel
* Parquet

## Data Sources

Data has been collected from a variety of sources, including:

* Open data portals (e.g., government datasets)
* Public APIs (exported snapshots)
* Web data (where permitted)

## Privacy & Ethics
* Only publicly accessible data is included
* No intentional inclusion of sensitive personal data
* If any dataset raises concerns, please open an issue
